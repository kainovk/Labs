# Задание:
### Создать класс Point — точка. На его основе создать классы ColoredPoint и Line. На основе класса Line создать класс ColoredLine и класс PolyLine — многоугольник.
### Все классы должны иметь методы для установки и получения значений всех координат, а также изменения цвета и получения текущего цвета.
### Написать демонстрационную программу, в которой будет использоваться список объектов этих классов в димнамической памяти.
